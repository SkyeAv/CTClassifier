# Clinical Trials Classifier

## Version 4.0.0 (Beta)

### By Skye Goetz, Gwênlyn Glusman, and Jackie Lindstrom

LightGBM Based Declarative GPU Accelerated Bionomial Classification Of Clinical Trial Data Equiped With Multiple Vizualizations And Exports. Based On AACT Database.

## Usage (With Nix)

Develop Nix Shell And Virtual Environment
```bash
nix develop -L . --impure
python3 -m venv venv/
source venv/bin/activate
python3 -m pip install -e .
```

Use Exposed CLI
```bash
CTC --help
```
